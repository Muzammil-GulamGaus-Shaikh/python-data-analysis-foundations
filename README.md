# 📊 Python Data Analysis Foundations

A structured Python practice repository focused on building strong logical foundations required for Data Analyst roles.

---

## 🎯 Objective

This repository is designed to strengthen:

- Core programming logic  
- Data aggregation techniques  
- Conditional filtering  
- Dictionary-based frequency analysis  
- Data cleaning methods  
- Business-style data handling  
- NumPy fundamentals for numerical computing  

The focus is on writing logic manually (without relying on built-in shortcuts) to deeply understand how data processing works internally.

---

# 🔹 Level 1 – Core Python Logic

Covers:

- Manual total sales calculation  
- Average computation  
- Maximum and minimum finder  
- Even number filtering  
- Frequency counter using dictionary  
- String data cleaning using `.strip()`  

These exercises strengthen loops, conditions, and data structure understanding.

---

# 🔹 Level 2 – Analyst Thinking

Focuses on:

- Filtering transactions based on business conditions  
- Working with list of dictionaries  
- Applying conditional logic for real-world scenarios  

Example:
Filtering transactions where `amount > 1000`
